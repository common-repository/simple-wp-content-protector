== Description ==

**This Plugin will automatically protect the content of your WordPress site**

= Features =

1. Disables the Mouse Right Click
2. Disables CTRL+C - Copy
3. Disables CTRL+X - Cut
4. Disables CTRL+V - Paste
5. Disables CTRL+U - View Source
6. Disables CTRL+A - Select All
7. Disables CTRL+S - Save
8. Disables CTRL+P - Print
9. Disables Image Dragging


== Installation ==

1. Upload `simple-wp-content-protector` Folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
